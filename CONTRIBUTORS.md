## How to Contribute

Contributions are welcome! Here are a few ways to get involved:

- **Bug reports** — open an issue describing the problem, your OS, shell version, and whether you use `git`, `jj`, or both
- **Feature requests** — open an issue with the `enhancement` label
- **Pull requests** — fork the repo, make your changes in a branch, and open a PR against `main`

Please keep pull requests focused on a single change. For larger changes, open an issue first to discuss the approach.

### Guidelines

- The script must remain a single self-contained Bash file with no external dependencies beyond `bash`, `find`, `awk`, `git`, and optionally `jj`
- Test against both macOS and Linux before submitting
- Follow the existing code style and keep the output format consistent
